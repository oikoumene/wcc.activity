wcc.activity Installation
-------------------------

To install wcc.activity using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``wcc.activity`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        wcc.activity

* Re-run buildout, e.g. with:

    $ ./bin/buildout
